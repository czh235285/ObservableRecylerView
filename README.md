# ObservableRecylerView
自定义上拉隐藏下拉显示的RecylerView

use Gradle:

```
repositories {
  maven { url "https://jitpack.io" }
  mavenCentral()
  google()
}
dependencies {
    implementation 'com.github.czh235285:LikeView:1.0.1'
}
```
